Información de la Aplicación

Propósito
Sistema para crear y gestionar informes de evolución de enfermería en Uci de forma rápida y eficiente.
Guardado Automático
Los datos se guardan automáticamente mientras escribes. Los informes generados se almacenan en tu navegador local.
Recuperar Informes
Puedes recuperar informes anteriores usando el desplegable "Seleccionar Informe Guardado". Útil para consultar o reimprimir informes previos.
Impresión por Turnos
• Imprimir: Imprime según tu turno actual
• Imprimir en Turno Alternativo: Cambia el texto del turno para imprimir informes del turno opuesto
Importante: Los datos solo están disponibles en este navegador y ordenador. Si cambias de dispositivo o borras datos del navegador, perderás la información guardada.
